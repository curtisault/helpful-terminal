# Standard Tools

These are available out of the box on unix.

Man pages can be found by typing

```
man {tool}
```

## Tool

1. man
2. !!
3. pwd
4. open
5. history
6. cat
7. wc
8. less/more
9. mv/cp
10. find
11. grep
12. piping
13. sed/awk
14. vi

```
df -h
```

```
grep -R 'client_id' lib | wc -l
```

## Local Machine

1. df
2. ps
3. top
4. pgreg
5. pkill

```
top -o rsize
```

## Networking

1. traceroute 
2. dig

```
traceroute main.crowbar.us-east-1.stage.corvusinsurance.com
```
