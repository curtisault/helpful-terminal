# Standard Tools

These are available out of the box on unix.

Man pages can be found by typing

```
man {tool}
```

## Tool

1. man
2. !!
3. df
4. pwd
5. history
6. cat
7. wc
8. less/more
9. mv/cp
10. find
11. grep
12. piping
13. sed/awk
14. ps
15. pgrep
16. pkill
17. vi

```
df -h
```
