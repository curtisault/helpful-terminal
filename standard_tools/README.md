# Standard Tools

These are available out of the box on unix.

Man pages can be found by typing

```
man {tool}
```

## Basic

1. history
2. tail
3. !!
4. pwd
5. open
6. man
7. cat
8. wc
9. less/more
10. mv/cp
11. find
12. grep
13. piping
14. sed/awk
15. diff

```
df -h
```

```
grep -R 'client_id' lib | wc -l
```

## Local Machine

1. df
2. ps
3. top
4. pgreg
5. pkill

```
top -o rsize
```

## Networking

2. dig

traceroute main.crowbar.us-east-1.stage.corvusinsurance.com
