# Standard Tools

These are available out of the box on unix.

Man pages can be found by typing

```
man {tool}
```

## Tool

1. man
2. !!
3. df
4. pwd
5. open
6. history
7. cat
8. wc
9. less/more
10. mv/cp
11. find
12. grep
13. piping
14. sed/awk
15. vi

```
df -h
```

```
grep -R 'client_id' lib | wc -l
```

## Local Machine

1. ps
2. top
3. pgreg
4. pkill

```
top -o rsize
```

## Networking

1. traceroute 
2. dig

```
traceroute main.crowbar.us-east-1.stage.corvusinsurance.com
```
